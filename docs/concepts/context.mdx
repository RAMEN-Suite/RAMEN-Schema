:::tip
Refinement should not make a project model **unnecessarily granular**. Internal textual structures, such as tables, table cells, or line groups, can often remain encoded and annotated within established formats such as [TEI](https://tei-c.org/).

They should become separate `Collection` or `Content` nodes only when they need to be addressed, linked, validated, or published as independent editorial objects.
:::
